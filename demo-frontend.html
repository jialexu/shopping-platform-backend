🛒 Shopping Service Live Demo

Full-Stack Microservices Architecture with Spring Boot & Multiple Databases

Spring Boot 3.3.4 Spring Cloud Gateway JWT Security MySQL MongoDB Cassandra Kafka Events Docker JUnit + Mockito
🔗 API Gateway: http://localhost:8080/api
📚 Swagger UI: http://localhost:8080/swagger-ui.html
🎛️ Kafka UI: http://localhost:8082

1 User Registration (Spring Data JPA + MySQL)

2 Authentication (JWT Token + Spring Security)

3 Product Catalog (MongoDB Document Store)

4 Inventory Management (Cassandra)

5 Order Creation (OpenFeign + Service Integration)

6 Payment Processing (Idempotency + Kafka Events)

7 Event-Driven Updates (Apache Kafka)

After payment, Kafka events trigger automatic order status updates
